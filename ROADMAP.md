## P0 · MVP (本期目标)

**交付物**：Windows `.exe` 安装包 v0.1.0，朋友双击即用。

**功能清单**：

- [ ] Electron + Vue 3 + TypeScript 项目骨架
- [ ] B 站弹幕协议接入（blive-message-listener）
- [ ] 平台适配层抽象 (PlatformAdapter 接口 + BilibiliAdapter 实现)
- [ ] 事件总线 + 规则引擎 + 动作派发器
- [ ] 三套默认规则：欢迎 / 关键词回复（仅 TTS+日志，不发弹幕）/ 礼物感谢
- [ ] Edge TTS 集成（Electron 主进程播放，5-8 个中文女声）
- [ ] 内嵌 Express + Socket.IO 服务，端口冲突自动避让
- [ ] OBS Overlay 单页：礼物弹出特效 + 进房欢迎条
- [ ] 控制台 UI：房间号配置 / 规则编辑 / TTS 设置 / 实时日志 / Overlay URL 复制按钮
- [ ] electron-builder 打包成 NSIS 安装器
- [ ] 主播使用说明 `docs/主播使用说明.md`

**工程量估算**：10-15 小时（Claude Code）

**验收标准**：

朋友拿到 `LiveLink-Setup-0.1.0.exe`：

1. 双击安装 → 启动应用 → 界面出现
2. 输入直播间号（例如 `21452505`） → 点"开始"
3. 状态栏显示"已连接"
4. 在 OBS 添加浏览器源 → 粘贴控制台显示的 Overlay URL → 礼物特效区出现
5. 直播间有人进来 → TTS 念出欢迎话术，OBS 上飘出欢迎条
6. 直播间收到礼物 → TTS 念感谢，OBS 上飘出礼物特效
7. 关闭应用 → 二次启动 → 房间号 / 规则全部记住

任何环节失败 → 主播界面上有清楚的错误提示和修复指引，不是 Electron Console 的 Stack Trace。

## P1 · 增强（待 P0 完工后启动）

## P2 · 智能化（待 P1 完工后启动）

**功能清单**：

- [ ] AI 智能回复（Claude API 接入，主播可选启用，需配 key）
  - 规则引擎里新增 "AI Match" 类型规则：弹幕送 LLM 判断意图 → 生成回复 → TTS 念出
  - 安全护栏：限速、敏感词过滤、用量监控
- [ ] 互动小游戏框架（一个游戏 = 一个 plugin）：
  - [ ] 投票（"!投A" / "!投B" → 实时统计 → Overlay 柱状图）
  - [ ] 赛马（弹幕选号 → 倒计时 → 随机赛跑动画 → 公布赢家）
  - [ ] 喂宠物（虚拟宠物，礼物 → 喂食 → 等级提升）
- [ ] 游戏热加载机制（不重启应用切换游戏）

**预估**：12-18 小时
