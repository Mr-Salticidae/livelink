# LiveLink 使用说明（主播朋友请看这一份就够了）

> 这是一个跑在你电脑上的小工具，连上你的 B 站直播间，自动欢迎进房观众、感谢礼物，再在你的 OBS 上飘出特效。
> 不用装 Python、不用敲命令、不用账号密码。

---

## 第一次使用：5 分钟搞定

### 1. 安装

1. 找到 `LiveLink-Setup-0.1.0.exe`，**双击**它。
2. 看到安装向导，**一路下一步**。可以改安装位置，也可以默认。完成。
3. 桌面会有一个 LiveLink 图标，双击启动。

第一次启动如果 Windows 弹了**蓝色的"已保护你的电脑"**窗口：

> 这是 Windows Defender 看到一个没花钱买"代码签名证书"的小软件，会习惯性提醒。
> 点 **"更多信息"** → 再点 **"仍要运行"** 就行。
> LiveLink 没有任何上传行为，全部数据都在你本地，可以放心。

### 2. 启动 + 连上你的直播间

应用启动后看到一个深色界面，左边有几个图标。

1. 在**首页**填上你的**直播间号**。
   - 怎么找？打开你 B 站直播间的网页，地址栏看 `https://live.bilibili.com/` 后面那串数字就是。
   - **直接把整段链接粘进去也行**，工具会自动抠出号码。
2. 点**绿色的"开始"**按钮。
3. 看到状态变成**"已连接"**，就成功了。

### 3. 把特效接进 OBS

1. 在首页，找到 "OBS 浏览器源 URL" 那一行。点 **"复制"**。
2. 打开你的 OBS Studio。
3. 在"来源"面板点 **+ 号** → 选 **"浏览器"**。
4. 起个名字（比如 "LiveLink Overlay"）→ 确定。
5. 把刚才复制的 URL **粘到 "URL" 那一栏**。
6. 把宽度填 **1920**、高度填 **1080**，确定。

完成。OBS 画面上现在多了一个透明层，等直播间一有动静（有人进来、有人送礼物），上面就会飘出特效。

## 平时怎么用

### 改欢迎话术 / 礼物感谢话术

左边点 **"规则"**。你会看到三条默认规则：

- **默认欢迎** —— 有人进直播间时念出来
- **问候关键词回复** —— 弹幕里出现 "你好""哈喽" 时念
- **默认礼物感谢** —— 有人送礼物时念

点某条规则的**"编辑"**，里面有 "TTS 模板"：

```
欢迎{uname}来到直播间
```

`{uname}` 是占位符，会被替换成观众的用户名。改完点**保存**就生效。

可用的占位符（不同事件能用的不一样）：

| 占位符 | 含义 | 哪些事件能用 |
|---|---|---|
| `{uname}` | 用户名 | 所有 |
| `{giftName}` | 礼物名 | 礼物 |
| `{num}` | 数量 | 礼物 |
| `{content}` | 弹幕内容 | 弹幕 / SC |
| `{guardLevel}` | 大航海等级 | 上舰 |

### 改 TTS 声音

左边点 **"TTS"**：

- **TTS 语音播报**：总开关
- **音色**：7 种可选（女声、男声、童声）。我（跳蛛先生）默认给你设的是 "晓晓"。
- **语速 / 音量**：拖滑条。改完自动保存。
- **测试声音**：随时按一下听效果。

### 看看现在都发生了什么

左边点 **"日志"**。这里实时滚动显示弹幕、礼物、进房等事件。
点上面的 **"弹幕 / 礼物 / 进房"** 切换分类查看。

## 出问题了怎么办

### 连不上直播间

界面上会有一行红字告诉你具体是什么情况：

- **"找不到这个直播间，或者主播还没开播"** —— 检查房间号有没有填错；或者你还没开播，开播之后再点开始也行。
- **"网络好像断了"** —— 检查 WiFi / 网线，路由器重启一下。
- **"房间号要填数字"** —— 输入框里一片空白时点开始会出现。填数字。

### TTS 没声音

1. 进 TTS 设置页，看 "TTS 语音播报" 开关是不是开的。
2. 点 "测试声音"，能听到说明 TTS 本身没问题——那就是 OBS 那边没把声音录到。
3. 如果点测试也没声，可能是网络问题（TTS 要联网到微软的服务器合成）。换个网络再试。

### OBS 上看不到特效

1. 确认 OBS 的浏览器源 URL 跟首页**完全一致**（端口可能跟默认 38501 不一样，工具会自动避让，复制最新的）。
2. 在 OBS 浏览器源属性里，点一下 "刷新缓存"。
3. 检查浏览器源的"宽 1920 高 1080"是不是设对了。
4. 看看应用的状态徽章是不是 "已连接"。

### 礼物送了但没特效

特效是收到 `SEND_GIFT` 事件才弹。常见原因：

- 你刚开始连接的前 1-2 秒，可能漏掉早到的事件。等几秒再观察。
- 银瓜子礼物（免费小心心）也算礼物，但没价格。如果想只感谢付费礼物，去**规则**页给"默认礼物感谢"的匹配条件改成"按价格筛选"——MVP 暂不支持，下个版本（P1）补。

## 其它你可能想知道的

- **数据存哪？** 配置和规则存在 `%APPDATA%/livelink-config.json`。卸载应用如果想顺便清理，删这个文件即可。
- **能不能让它发弹幕回应观众？** MVP 不行（要登录账号，有风险）。下个版本（P1）会评估开放。
- **支持抖音 / 虎牙吗？** 暂时只 B 站。架构层留好了，未来可加。
- **要付费吗？需要 API key 吗？** 不要。一切免费、本地、零密钥。
