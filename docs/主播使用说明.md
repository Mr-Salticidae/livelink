# LiveLink 使用说明（主播朋友请看这一份就够了）

> 这是一个跑在你电脑上的 B 站直播助手。
> 基础：自动欢迎进房观众、感谢礼物、OBS 上飘出特效。
> 进阶：弹幕抽奖、互动投票、赛马小游戏、SuperChat 横幅、盲盒查询、主播弹幕悬浮窗（钉住穿透不挡游戏）、给观众看的弹幕信息板。
> 不用装 Python、不用敲命令、不用账号密码。**v1.0 · 一切免费、本地、零密钥**。

---

## 第一次使用：5 分钟搞定

### 1. 安装

1. 找到 `LiveLink-Setup-1.0.0.exe`（文件名后面那串数字是版本号，跳蛛先生给你的版本可能不一样），**双击**它。
2. 看到安装向导（向导是简体中文的），**一路下一步**。可以改安装位置，也可以默认。完成。
3. 桌面会有一个 LiveLink 图标，双击启动。开始菜单也会有一个 **LiveLink** 文件夹，里面同时放了"LiveLink"（启动）和"Uninstall LiveLink"（卸载）两个快捷方式。

第一次启动如果 Windows 弹了**蓝色的"已保护你的电脑"**窗口：

> 这是 Windows Defender 看到一个没花钱买"代码签名证书"的小软件，会习惯性提醒。
> 点 **"更多信息"** → 再点 **"仍要运行"** 就行。
> LiveLink 没有任何上传行为，全部数据都在你本地，可以放心。

### 2. 启动 + 连上你的直播间

应用启动后看到一个深色界面，左边有几个图标。

1. 在**首页**填上你的**直播间号**。
   - 怎么找？打开你 B 站直播间的网页，地址栏看 `https://live.bilibili.com/` 后面那串数字就是。
   - **直接把整段链接粘进去也行**，工具会自动抠出号码。
2. 点**绿色的"开始"**按钮。
3. 看到状态变成**"已连接"**，就成功了。

### 3. 把特效接进 OBS

1. 在首页，找到 "OBS 浏览器源 URL" 那一行。点 **"复制"**。
2. 打开你的 OBS Studio。
3. 在"来源"面板点 **+ 号** → 选 **"浏览器"**。
4. 起个名字（比如 "LiveLink Overlay"）→ 确定。
5. 把刚才复制的 URL **粘到 "URL" 那一栏**。
6. 把宽度填 **1920**、高度填 **1080**，确定。

完成。OBS 画面上现在多了一个透明层，等直播间一有动静（有人进来、有人送礼物），上面就会飘出特效。

## 平时怎么用

### 改欢迎话术 / 礼物感谢话术

左边点 **"规则"**。你会看到三条默认规则：

- **默认欢迎** —— 有人进直播间时念出来
- **问候关键词回复** —— 弹幕里出现 "你好""哈喽" 时念
- **默认礼物感谢** —— 有人送礼物时念

点某条规则的**"编辑"**，里面有 "TTS 模板"：

```
欢迎{uname}来到直播间
```

`{uname}` 是占位符，会被替换成观众的用户名。改完点**保存**就生效。

可用的占位符（不同事件能用的不一样）：

| 占位符 | 含义 | 哪些事件能用 |
|---|---|---|
| `{uname}` | 用户名 | 所有 |
| `{giftName}` | 礼物名 | 礼物 |
| `{num}` | 数量 | 礼物 |
| `{content}` | 弹幕内容 | 弹幕 / SC |
| `{guardLevel}` | 大航海等级 | 上舰 |

### 改 TTS 声音（含分事件音色）

左边点 **"TTS"**：

- **TTS 语音播报**：总开关
- **音色**：8 种可选（4 女声 + 4 男声，含东北话 / 陕西话方言）。默认晓晓
- **语速 / 音量**：拖滑条。改完自动保存
- **测试声音**：随时按一下听效果

**分事件音色（多角色）**：页面下半部分有"分事件音色"区。可以给每类事件单独设音色：
- 欢迎用萌妹声、礼物用甜美、SC 用沉稳新闻播音员……
- 选"使用全局"就沿用上面的全局音色
- 每行有"试听"按钮，直接听该事件实际会用的声音

主播不看屏幕也能从音色辨别是哪类事件。

### 看看现在都发生了什么

左边点 **"日志"**。这里实时滚动显示弹幕、礼物、进房等事件。
点上面的 **"弹幕 / 礼物 / 进房"** 切换分类查看。

## 进阶功能（v1.0 新增）

### 主播弹幕悬浮窗（单屏主播必备）

游戏全屏时也能瞟一眼弹幕，不用切回 OBS。

1. **首页 → "弹幕悬浮窗"** 卡片，打开开关 → 屏幕角落弹出深色小窗
2. 拖到舒服位置 / 缩放大小（自动记忆下次启动）
3. 标题栏右上的 📌 图标 → 点一下进入**钉住模式**：
   - 不可拖动 / 缩放
   - **鼠标完全穿透**（游戏正常操作，不会误触把它拖走）
   - 不抢游戏焦点
4. 钉住后子窗 📌 按钮也被穿透，**解开只能去 Home 页同位置开关**
5. 标题栏中段有 `👁 1.2 万` 蓝色徽章实时显示直播间观看人数
6. 新弹幕进场时**黄色高亮 → 2.4s 淡出**，礼物用更暖的金色
7. 全屏游戏遮挡问题：如果你玩**独占全屏游戏**，OS 层任何 alwaysOnTop 都覆盖不了（Discord/Steam overlay 都靠 D3D hook 实现，本工具不做）→ 把游戏画面切到 **"无边框窗口"** 模式，多数游戏都有这选项

### 直播屏弹幕信息板（给观众看）

让弹幕在直播画面上一直滚动给观众看（OBS overlay）。

1. **首页 → "直播屏弹幕信息板"**，打开开关
2. 配置：
   - **位置**：拖动 16:9 预览框里的"弹幕板"缩略图到屏幕任意位置；下面有 4 角快速预设
   - **最多显示条数**（5-30）/ 字号（12-24）
   - **礼物事件也进信息板**（金色显示）
3. OBS 浏览器源会自动同步（不用手动 reload）

### 弹幕抽奖

侧边栏 🎰 **抽奖**：

1. 配置：奖品 / 关键词 / 中奖人数 / 倒计时 / 可选粉丝牌门槛
2. 点"开始抽奖" → Overlay 弹大字"🎰 抽奖开始啦"招牌 → 进入倒计时
3. 观众弹幕**包含**关键词 → 自动报名（uid 去重）
4. 倒计时到 / 点"立即开奖" → Fisher-Yates 抽取 → Overlay 弹结果卡 + 飘彩庆祝 10 秒
5. 主窗显示完整中奖名单，可"复制名单"

### 互动投票

侧边栏 📊 **投票**：

1. 配置：标题 / 2-6 选项（key + 名称） / 倒计时 / 是否允许改投
2. 启动后 Overlay 弹"📊 投票时间"招牌 + 实时柱状图
3. 观众弹幕**完全等于** key（"1" / "A"）才计一票（避免"1 毛"被误计）
4. 倒计时到 → Overlay 弹结果卡（赢家 ★ 高亮）+ 飘彩 9 秒

### 赛马

侧边栏 🐎 **赛马**：

1. 配置：2-8 匹马（号 + emoji + 马名）/ 报名时长 / 赛跑时长上限
2. 启动 → Overlay 弹"🏇 赛马来啦"招牌 → 进入报名期
3. 观众弹幕发马号押注（同 uid 改号覆盖前一次）
4. 报名结束 / 立即开赛 → 水平赛道动画，每 200ms 推进，6% 加速 / 4% 失蹄
5. 第一匹到终点 → 公布 🥇🥈🥉 排名 + **押中冠军的观众名单** + 飘彩 10 秒

### SuperChat 横幅

收到醒目留言会自动弹横幅（不用配置），按价位 4 档视觉：
- < 100 元：蓝紫渐变简洁卡 / 顶部居中
- 100-499：紫色光晕脉冲
- 500-999：金色强光晕脉冲
- ≥ 1000：红金渐变 + 屏幕中央 + 10 粒子放射 + 大字号

TTS 也会自动念出（"感谢 XXX 的 X 元醒目留言"，可在规则页改话术）。

### 盲盒查询

观众送过盲盒后，任何观众发"查盲盒 / 查盈亏 / 我的盲盒" → Overlay 屏幕中央弹卡片 7 秒，显示该观众累计花费 / 中奖 / 净盈亏 + 最近 5 条详细记录。数据按房间隔离持久化。

### 礼物特效按价位分档

主播什么都不用配，礼物来了 overlay 自动按价位 3 档显示：
- silver / 0 元：普通飞入动画
- 1-30 元：放大缩放 + 金色光晕
- > 30 元：屏幕中央大字 + 8 粒子放射

特效图是从 B 站礼物配置 API 拉的真实礼物图，第一次见的礼物自动缓存到本地。

### 粉丝牌过滤（欢迎规则）

进规则页编辑"默认欢迎"，**匹配方式**下拉选 **"粉丝牌等级"**：
- 最低粉丝牌等级（0-40）
- 必须是本主播的粉丝牌

**caveat**：B 站协议层 INTERACT_WORD 进房事件**很少携带粉丝牌信息**，只有送过礼物 / 发过弹幕的老粉再次进房才会被识别。新观众纯进房通常没 badge。

## 出问题了怎么办

### 连不上直播间

界面上会有一行红字告诉你具体是什么情况：

- **"找不到这个直播间，或者主播还没开播"** —— 检查房间号有没有填错；或者你还没开播，开播之后再点开始也行。
- **"网络好像断了"** —— 检查 WiFi / 网线，路由器重启一下。
- **"房间号要填数字"** —— 输入框里一片空白时点开始会出现。填数字。

### TTS 没声音

1. 进 TTS 设置页，看 "TTS 语音播报" 开关是不是开的。
2. 点 "测试声音"，能听到说明 TTS 本身没问题——那就是 OBS 那边没把声音录到。
3. 如果点测试也没声，可能是网络问题（TTS 要联网到微软的服务器合成）。换个网络再试。

### OBS 上看不到特效

1. 确认 OBS 的浏览器源 URL 跟首页**完全一致**（端口可能跟默认 38501 不一样，工具会自动避让，复制最新的）。
2. 在 OBS 浏览器源属性里，点一下 "刷新缓存"。
3. 检查浏览器源的"宽 1920 高 1080"是不是设对了。
4. 看看应用的状态徽章是不是 "已连接"。

### 礼物送了但没特效

特效是收到 `SEND_GIFT` 事件才弹。常见原因：

- 你刚开始连接的前 1-2 秒，可能漏掉早到的事件。等几秒再观察。
- 银瓜子礼物（免费小心心）也算礼物，但没价格。如果想只感谢付费礼物，去**规则**页给"默认礼物感谢"的匹配条件改成"按价格筛选"——MVP 暂不支持，下个版本（P1）补。

### 礼物连击时 TTS 怎么播

礼物连击场景下，工具会**按到达顺序逐条播报**：第 1 条念完才念第 2 条，依次类推。
TTS 队列上限是 20 条，连击太密时超过 20 条的最早那几条会被丢弃（避免主播在念几分钟前的礼物）。

下一个版本（P1）会做"窗口内合并播报"——同一人连击的礼物自动合成一条"感谢 XXX 送出 N 个 X"，避免重复念。当前 MVP 是逐条播。

## 其它你可能想知道的

- **数据存哪？** 配置和规则存在 `%APPDATA%/livelink-config.json`。**卸载只会移除 LiveLink 应用程序本身，不会动这个配置文件**（方便你下次重装时自动接回房间号、规则、TTS 偏好等）。想彻底清干净见下面 →
- **B 站登录信息（SESSDATA）安全吗？** SESSDATA 在保存到 `livelink-config.json` 之前会被 Windows 系统级加密（DPAPI，跟你当前 Windows 账号绑定），**把这个文件复制到别的电脑也解不开**。所以即使有人拿到了你的配置文件，也用不了你的 B 站账号。
- **能不能让它发弹幕回应观众？** MVP 不行（要登录账号，有风险）。下个版本（P1）会评估开放。
- **支持抖音 / 虎牙吗？** 暂时只 B 站。架构层留好了，未来可加。
- **要付费吗？需要 API key 吗？** 不要。一切免费、本地、零密钥。

## 想彻底清干净（卸载 + 删配置）

正常卸载只移除 LiveLink 应用程序本身，**不会删你存的房间号 / 规则 / TTS 偏好 / B 站登录信息**——这是为了你下次重装时配置自动接回。

如果你想彻底清干净（比如换台电脑、或者主播圈分享配置给朋友），按这两步：

1. **正常卸载 LiveLink**：开始菜单 → LiveLink 文件夹 → **Uninstall LiveLink**（或者控制面板的"程序和功能"里找 LiveLink）
2. **删配置文件**：按 `Win + R`，输入 `%APPDATA%`，回车，找到 `livelink-config.json` 删掉

或者 PowerShell 一行搞定：

```powershell
Remove-Item "$env:APPDATA\livelink-config.json" -ErrorAction SilentlyContinue
```

⚠️ 注意：配置文件里的 SESSDATA 虽然是加密的（跟当前 Windows 账号绑定，别的电脑解不开），但**配置文件还是不建议直接发给别人**——里面的房间号、规则等信息你可能也不想分享。要分享配置时，建议先把 SESSDATA 那一栏清空再发。
