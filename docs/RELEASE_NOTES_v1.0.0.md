## ✨ v1.0 全部功能

### 基础（开箱即用）

- 自动欢迎进房观众（TTS + OBS 飘条）
- 弹幕关键词回复（默认匹配 你好 / 哈喽 / hi / hello）
- 礼物 TTS 感谢 + OBS 屏幕特效
- 实时事件日志（进房 / 弹幕 / 礼物 / SC / 上舰 滚动显示）
- 网络抖动后断线自动重连（5s → 15s 两轮）
- B 站登录态本地 DPAPI 加密存储（跨电脑解不开）

### 礼物 & SuperChat

- 礼物特效按价位 3 档：0 元普通飞入 / 1-30 元金色光晕 / >30 元中央 8 粒子放射
- 真实礼物图（从 B 站礼物配置 API 拉，首次见的自动缓存）
- SuperChat 横幅 4 档视觉：蓝紫 / 紫光晕 / 金光晕 / 红金粒子（≥1000 元）
- SC 自动 TTS 念出

### 弹幕互动小游戏

- 🎰 **弹幕抽奖**：关键词触发参与 → 倒计时 → Fisher-Yates 抽取 → 结果卡 + 飘彩
- 📊 **互动投票**：弹幕严格匹配 key 计票 → 实时柱状图 → 赢家高亮
- 🐎 **赛马**：2-8 匹马 / 弹幕押注 / 水平赛道动画 / 公布冠军 + 押中观众名单

### 主播工具

- 主播弹幕悬浮窗（钉住后鼠标穿透不挡游戏）
- 观看人数实时徽章（悬浮窗标题栏）
- 新弹幕黄色高亮 2.4s 淡出 / 礼物用更暖的金色
- 直播屏弹幕信息板（给观众看的滚动弹幕，可拖动到任意位置）

### TTS 进阶

- 8 种中文音色稳定可用（4 女 + 4 男，含东北话 / 陕西话方言）
- 分事件音色（多角色）：欢迎用萌妹 / 礼物用甜美 / SC 用新闻播音
- 语速 / 音量随意调，每事件可独立试听

### 规则与查询

- 3 套默认规则的文字模板可编辑（欢迎话术 / 关键词 / 礼物感谢）
- 首页快捷开关（一键关闭欢迎 / 关键词回复 / 礼物感谢）
- 粉丝牌过滤（欢迎规则可按最低粉丝牌等级 + 必须本主播）
- 🎁 盲盒盈亏查询（观众发"查盲盒"→ Overlay 弹该观众的累计盈亏卡）

## 🚧 已知限制

| 项 | 说明 |
|---|---|
| Windows SmartScreen 蓝屏 | 未签名应用首次启动会弹"已保护你的电脑"——点 **更多信息 → 仍要运行** 即可。本应用 100% 本地运行，无任何上传。 |
| 弹幕需要 SESSDATA | B 站从 2023 年起对未登录连接限制了 DANMU_MSG 推送。首页 → 高级·B 站登录态 → 填 SESSDATA + DedeUserID，详见使用说明 §2 |
| 暂仅支持 B 站 | 抖音 / 虎牙的协议适配在路线图但未排期 |
| 暂不发弹幕到直播间 | 涉及账号风险，后续评估 |
| 粉丝牌过滤覆盖率有限 | B 站协议层 INTERACT_WORD 进房事件**很少携带粉丝牌**，只有送过礼物/发过弹幕的老粉再进房才识别 |

## 📌 系统要求

- **Windows 10 / 11**（macOS / Linux 暂不支持）
- 4 GB RAM 起步（应用本身约占 350 MB）
- 联网（弹幕监听 + TTS 合成需要网络）
- 推荐配合 OBS Studio 27+ 使用

## 📜 法律声明

LiveLink 是**非官方**第三方工具，仅供个人主播使用，**与哔哩哔哩公司无关**。

- 仅读取直播间公开推送数据（弹幕 / 礼物 / 进房等）

- 配置文件 `%APPDATA%/livelink-config.json` 可随时手动删除
